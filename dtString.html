A string in javascript must be surrounded by quotes.
In javascript , there are 3 types of quotes.

  1. Double quotes :"hello"
  2. Single quotes: 'hello'
  3. Backticks:`hello`

        code 1:
        let str="Hello";
        let str2='single quotes';
        let str3=`can embed another ${str} ${3+4}`;
        //Backticks are "extended functionality" quotes. They allow us to embed variable and expression into a string by wrapping them in ${...}
        alert(str);
        alert(str2);
        alert(str3);
