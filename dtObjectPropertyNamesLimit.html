As we already know , a variable cannot have a name eual to one of
language -reserved word like "for", "let","return" etc.

But for an object property, there's no such restriction:

        code 1:
            let obj={
                for:2,
                let:8,
                return:2
            };
    
            alert(obj.for+obj.let+obj.return);//12
        
    
        code 2:
            //Other types are automatically converted to strings.
            let object={
                0:"test",//Same as "0":"test"
                name:"op"
            };
    
            //Both alerts access the same property(the names 0 is converted to string "0")
            alert(object["0"]);//test
            alert(object[0]);//test(same property)
            alert(object[name]);//undefined
