String conversion

String conversion happens when we need to string form of a value.
String (value) function to convert a value to a string. 

 code 1:
        let value = true;
    alert(typeof value);//boolean

    value =String(value);//now value is a string "true"
    alert(typeof value);//string

Numeric conversion

Numeric conversion is mathematical function and expressions happens automatically .
Value Becomes...
undefined NaN
null 0
True or False 1 or 0
String Whitespaces (include spaces, tabs \t ,newlines \n etc.) from the start and end are removed.If the remaining string is empty, the result is 0. otherwise,the number is "read" from the string. An error give NaN. 

 code 2:
        let str="123.76";//string type
        alert(typeof str);//string
        
        str=Number(str);
        alert(typeof str);//number Number function convert either int or float from string
        alert(str);//123.76
