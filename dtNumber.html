For number of any kind:
integer and floating-point both in number data type.

"Special numeric values" which also belong to number data type:

  1. Infinity
  2. -Infinity
  3. NaN

example:
alert(1/0);//Infinity
alert(Infinity);//Infinity
alert(-Infinity);//-Infinity
//NaN (not a number)
alert("not a number"/2);//NaN
alert(3*NaN);//NaN
//exception
alert(NaN ** 0);//1
